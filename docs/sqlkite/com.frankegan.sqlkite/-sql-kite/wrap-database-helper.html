Wrap a SupportSQLiteOpenHelper for observable queries.

While not strictly required, instances of this class assume that they will be the only ones interacting with the underlying SupportSQLiteOpenHelper and it is required for automatic notifications of table changes to work. See the for more information on that behavior.
